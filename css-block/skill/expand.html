# 扩大点击区域

在不影响布局的前提下，扩大可点击的区域。


.expand {
    position: relative;
}
.expand:before {
    content: '';
    position: absolute;
    top: -15px;
    right: -15px;
    bottom: -15px;
    left: -15px;
}
